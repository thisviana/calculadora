Calculadora Simples

Projeto de **estudo** que implementa uma calculadora simples no terminal usando **Shell Script** e **Python**.

Objetivo
Praticar lógica de programação, entrada de dados pelo terminal e estruturas condicionais.

Tecnologias
- Shell Script
- Python 3

Como executar o arquivo `Calculadora.sh`

1. Abra o terminal no diretório onde está o arquivo `calculadora.sh`.
2. Conceda permissão de execução ao script:
chmod 744 calculadora.sh

Explicação do código em Python

O código em Python funciona da seguinte forma:
Exibe uma mensagem inicial no terminal.
Solicita dois números ao usuário e converte os valores para o tipo float.
Solicita a operação matemática desejada (+, -, *, /).
Utiliza estruturas condicionais (if, elif, else) para identificar a operação escolhida.
Realiza o cálculo correspondente.
Verifica se há tentativa de divisão por zero.
Exibe o resultado final no terminal.
